iCal

Plugin Identifier: 20000

Summary

Event

iCal Create event

eventId:=iCal Create event (calendar; startDate; endDate; properties; values)

calendar (TEXT)

Name (title) or UID of calendar to register the event.

startDate (TEXT)

endDate (TEXT)

properties (ARRAY)

Optional. Any other property to set for the new event. Setting the calendar here will not over-write $1.

values (ARRAY)

eventId (TEXT)

Empty string means the event could not be created. More information may be available in the console.

Discussion

error:=iCal Get event property (eventId; propertyName; propertyValue; date)

propertyName (TEXT)

You can use a iCal Event Properties constant for propertyName.

propertyValue (TEXT)

date (TEXT)

Optional. Use to specify an instance of a recurring event.
